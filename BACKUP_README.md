# 📦 부부신호등 소스 코드 백업 가이드

지금까지 공들여 만든 어플의 **기능, 디자인, 로직**을 안전하게 보관하기 위한 백업 방법입니다.

## 1. 한 번에 백업하기 (스크립트 사용)
제공해 드린 `scripts/backup_app.sh` 파일을 실행하면 현재 프로젝트의 핵심 소스 코드만 골라서 압축 파일을 만들어줍니다.

- **실행 방법**: 터미널(Terminal)에서 아래 명령어를 입력하세요.
  ```bash
  sh scripts/backup_app.sh
  ```
- **결과**: `HeartSync_Backup_날짜_시간.zip` 형태의 파일이 생성됩니다.
- **포함 범위**: `src` (로직/디자인), `public` (이미지), `index.html`, `package.json` 등 핵심 소스 전체
- **제외 범위**: `node_modules` (라이브러리 덩어리), `.git` (이력), 개인 데이터(DB 데이터)

## 2. Git을 활용한 백업 (권장)
이미 프로젝트 폴더에 `.git` 설정이 되어 있습니다. 작업이 끝날 때마다 아래 명령어를 통해 변경 사항을 기록(Snapshot)할 수 있습니다.

```bash
git add .
git commit -m "작업 내용 요약 (예: 기도 타임라인 라벨 수정)"
```

## 3. 주의사항
- 이 백업은 **'프로그램의 설계도(코드)'**를 백업하는 것입니다.
- 실제 사용자가 입력한 데이터(기도 제목, 메모 등)는 **Supabase 클라우드 데이터베이스**에 안전하게 저장되어 있으므로, 코드를 백업한다고 해서 DB 데이터가 사라지거나 함께 백업되지는 않습니다. (DB는 Supabase 대시보드에서 별도로 관리됩니다.)

언제든 현재 상태를 영구적으로 보관하고 싶으실 때 1번의 스크립트를 실행해 주세요!
